# Screenshots

Drop the GUI screenshots referenced from the project README into this folder.

The README (and its localised copies) expect the following filenames — feel
free to add more, just keep them under ~500 KB each and aim for a 16:10 or
3:2 frame so the chart and sidebar don't look cramped.

| File                | Page in the GUI                          |
|---------------------|------------------------------------------|
| `overview.png`      | Overview dashboard (live telemetry)      |
| `fans.png`          | Fans — curve editor in Curve mode        |
| `led-animation.png` | LED Ring — Animation mode with presets   |
| `settings.png`      | Settings — Diagnostics group expanded    |

When updating: replace the file in place. README links are stable, so a
fresh PNG is enough — no markdown change required.
